# Parametric Value at Risk (VaR) Calculation

This project implements a Parametric (Variance-Covariance) Value at Risk model 
to estimate potential portfolio losses under normal distribution assumptions.

## Objective
To calculate portfolio VaR using volatility, correlation, and confidence levels.

## Key Concepts
- Value at Risk (VaR)
- Covariance matrix
- Portfolio volatility
- Z-score and confidence levels

## Tools & Libraries
- Python
- NumPy
- Pandas
- SciPy
- Matplotlib
